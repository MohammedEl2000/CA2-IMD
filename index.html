Project Overview

My project is about Infrastructure as Code (IaC) which is a very interesting topic to research and learn more about and it also brings multiple different technologies together. It would be a great experience to investigate this technology in more depth and see what it has to offer to different users and also try to experiment on it and what it is capable of doing. The main reason of choosing Infrastructure as Code as my Project topic was my internship that allowed me to explore this technology and get hands on experience with it. The project is going to be based on the 4 different Infrastructure as Code tools and what each tool is capable of and what are their benefits and use cases. I will be using the tools to experiment on using a template that will have some functionalities that need to be established and implemented. All the implementation and experiment will be conducted next semester as for this semester will just be all research and getting to know the tools in more detail.


Research For The Project:

The research for the project will be based on Infrastructure as Code in general and the 4 IaC tools which will be used to implement different infrastructures and and see what the differences between them are. I will be researching IaC in more depth and discovering what are the key points of IaC. Some key points that I came across would be:

  • The benefits of IaC
  • Advantages and Disadvantages of IaC
  • The difference between IaC and IaaS
  • How IaC works
  • The different IaC tool types

After I research IaC in more depth and gather enough information about IaC, I will start to research more about the four different tool that I intend using in this project. Each tool has it benefits and cons and each tool also has a different workflow on how it operates. The tools that I intend to test in this project are:

  • Terraform
  • AWS CloudFormation
  • Puppet
  • Chef

I decided to chose these four tool because they where the most familiar to me and when I conducted the first phase research, they seemed very interesting to me and they where the most popular in the market from which I read on different websites. Terraform and CloudFormation are provisioning tools while Puppet and Chef are configuration management tools. A deep research will be conducted on each tool.
